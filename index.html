Live Keyboard

The LiveKeyboard class implements a stylable canvas-based widget representing a piano keyboard.

Basic Usage; Default Appearance and Behavior

By default, the class creates a two-octave monophonic keyboard that takes the full width of its container, and adjusts its height accordingly

In the HTML, define a container that will hold the keyboard widget, and specify the width of the element. In the JavaScript, create a new instance of the LiveKeyboard class, passing the desired HTML container as a parameter

Example

HTML

<div id='kbd1' style='width: 200px'></div>

JavaScript


var kbd1container = document.getElementById('kbd1');
var kbd1 = new LiveKeyboard(kbd1container);

Result

Styling and Behavior

Styling and behavior options can be applied as the second argument given to the constructor when creating a new keyboard object, or with the setOptions( ) method.

Example

The following example demonstrates some available styling and behavior options

HTML


<div id='kbd2' style='width: 500px'></div>

JavaScript


var kbd2container = document.getElementById('kbd2');
var kbd2 = new LiveKeyboard(kbd2container, {
  bottomNote: 21,
  topNote: 95,
  whiteKeyColor: 'black',
  blackKeyColor: 'red',
  keyBorderColor: 'yellow',
  whiteKeyActiveColor: '#ff9988',
  blackKeyActiveColor: '#887711',
  mode: 'polyphonic',
  editable: true
});

Result

List of Available Options

  • bottomNote {number}
  • topNote {number}
  • whiteKeyColor {string}
  • blackKeyColor {string}
  • keyBorderColor {string}
  • whiteKeyActiveColor {string}
  • blackKeyActiveColor {string}
  • mode {string}
  • editable {boolean}

Methods

  • setBottomNote(midiPitch)
    parameter midiPitch - the MIDI pitch number for the desired bottom note
  • setTopNote(midiPitch)
    parameter midiPitch - the MIDI pitch number for the desired top note
  • setActiveChord(activeChord)
    parameter activeChord - an array of MIDI pitches to set the active chord
